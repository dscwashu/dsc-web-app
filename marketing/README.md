# marketing

## Demo

[Here is a website hosted on netlify](https://www.dscwashu.com)

## How to Run Website

1.  Clone this repo
2.  `cd` into this `marketing` directory
3.  Run `npm install` to install relevant npm modules
4.  Use `gatsby develop` to host website at `localhost:8000`
